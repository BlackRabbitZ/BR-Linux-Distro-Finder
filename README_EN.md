# BlackRabbitZ Linux Distributions Finder

## Overview

BlackRabbitZ Linux Distributions Finder is an advanced Linux recommendation platform with a modern black/red cyber-style GUI.

The application analyzes:
- Hardware
- Privacy requirements
- Security needs
- Gaming compatibility
- Linux experience
- Development workflows
- Virtualization needs
- Pentesting and cybersecurity interests

to intelligently recommend the most suitable Linux distributions.

---

## Features

- Modern BlackRabbitZ GUI design
- Full dark red/black cyber-style interface
- Intelligent Linux recommendation engine
- Professionally structured questionnaire
- Gaming-focused Linux recommendations
- Ethical hacking & pentesting recommendations
- Privacy & anonymity focused systems
- Development & programming environments
- Server & virtualization support
- Recommendations for old/low-end hardware
- Multiple-choice support
- Integrated scoring system
- Save analysis results
- Distribution overview window
- Integrated About window
- Automatic logo support
